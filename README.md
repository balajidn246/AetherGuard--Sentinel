AetherGuard Sentinel
Real-Time Threat Detection & SOC Intelligence Platform

AetherGuard Sentinel is a full-stack enterprise-grade SOC (Security Operations Center) and SIEM (Security Information and Event Management) platform built to simulate real-world cybersecurity monitoring and incident response operations.

The platform provides real-time log ingestion, threat detection, anomaly analysis, incident management, and live attack visualization through a modern SOC analyst dashboard inspired by commercial solutions like Splunk, IBM QRadar, and Microsoft Sentinel.
